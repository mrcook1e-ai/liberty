# Liberty Engine

Графический интерфейс для настройки и запуска средств обхода блокировок (Zapret) с интеграцией Discord для проверки работоспособности.

## Особенности
- Автоматическая распаковка необходимых бинарных файлов в `%APPDATA%\liberty`.
- Настройка Discord токенов и ID через интерфейс (сохраняются локально в `settings.json`).
- Проверка доступности регионов Discord через встроенного бота.
- Поддержка фикса финских серверов через редактирование `hosts`.

## Требования для сборки
Для компиляции проекта вам понадобятся:
1. **Go** (версия 1.21 или новее) — [golang.org](https://go.dev/dl/)
2. **Node.js** и **npm** — [nodejs.org](https://nodejs.org/)
3. **Wails CLI** — устанавливается командой:
   ```bash
   go install github.com/wailsapp/wails/v2/cmd/wails@latest
   ```
4. **WebView2 Runtime** (обычно уже есть в Windows 10/11).

## Сборка проекта

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/mrcook1e-ai/liberty.git
   cd liberty
   ```

2. **Соберите приложение:**
   Используйте команду Wails для автоматической сборки фронтенда и бэкенда в один исполняемый файл:
   ```bash
   wails build -o LibertyGUI.exe
   ```
   После завершения готовый файл появится в директории `build/bin/`.

## Технические детали
- **Backend:** Go (Wails v2)
- **Frontend:** React + Tailwind CSS
- **Рабочая директория:** По умолчанию `%APPDATA%\liberty`. Туда распаковываются `winws.exe`, `botcheck.exe` и драйверы.
- **Списки:** Находятся в подпапке `lists/` внутри рабочей директории.

## Разработка
Для запуска в режиме разработки с живой перезагрузкой (Hot Reload):
```bash
wails dev
```
